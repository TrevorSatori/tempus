# Install

> To install this repo there are several options, 

### Docker Install


### Build from source


1. Clone the repo
```sh
git clone https://github.com/TrevorSatori/tempus.git
```

2. change directory to the repo
```sh
cd tempus
```

3. Install the dependencies
```sh
npm install
```

4.build an optimized version of the project
```sh
npm npm run build
```

5. run and enjoy 
```sh
npm npm run start
```
